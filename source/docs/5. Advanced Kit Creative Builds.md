## 5.1 Smart Distance Meter

### 5.1.1 Project Overview

这是一把会 “实时测距” 的测距电子尺，将物体放在两个挡板之间时，点阵屏会实时显示超声波传感器测到的距离，就像一位精准的测量小助手，轻松完成长度检测任务。

<img src="../_static/media/chapter_5/section_1/media/image21.png"  class="inline-icon" style="width:400px;" >

### 5.1.2 Learning Objectives

1. 在作品中实践超声波传感器与点阵屏的基础功能，理解距离检测与数据显示的联动逻辑。
2. 掌握测距电子尺的机械搭建。
3. 学会编写 “超声波测距 + 点阵显示” 的程序，理解传感器数据处理与可视化原理。
4. 提升动手实践与数据应用能力，感受智能测量的科技魅力。

## 5.3 Reaction Speed Challenge

### 5.3.1 Project Overview

这是一台会 “考验手速” 的手速挑战机，点阵上电显示 “prep”，按键按下后点阵显示 10 秒倒计时，蜂鸣器发出提示音；倒计时期间，触摸传感器触发速度越快，手才不会被打到，一旦被打到或 10 秒结束，挑战失败并发出警示音，就像一场紧张的反应力游戏。

<img src="../_static/media/chapter_5/section_3/media/image1.png"  class="inline-icon" style="width:400px;" >

### 5.3.2 Learning Objectives

1. 在作品中实践舵机、触摸、按键、点阵与主控蜂鸣器的基础功能，理解倒计时、手速检测与失败判定的联动逻辑。
2. 掌握手速挑战机的机械搭建。
3. 学会编写 “按键启动 + 触摸手速检测 + 成败判定” 的完整程序，理解多设备协同工作原理。
4. 提升动手实践与互动游戏设计能力，感受反应力挑战的编程乐趣。

2. 掌握旋转大摆锤的机械搭建。
3. 学会编写 “按键触发 + 舵机摆动” 的程序，理解往复运动控制原理。
