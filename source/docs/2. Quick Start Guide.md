# 2. Quick Start Guide

## 2.1 ESP32 Controller Overview

### 2.1.1 Introduction

这是一款以ESP32为核心，支持图形化编程和Python编程的智能主控器。它采用PC塑料外壳封装，板载PWM舵机接口、电机接口、可编程按钮、蜂鸣器等电子模块，并预留多个传感器接口，可拓展性强，统一4PN防反插接口，适配幻尔全系列传感器，使用方便安全。

<img src="../_static/media/chapter_2/section_1/media/image1.png"  class="inline-icon" style="width:400px;" >

### 2.1.4 Battery Charging Notes

1. 确保主控器的开关拨到“**OFF**”档。将电池安装在主控器的的电池槽内。（注意：正负极不要接反）

   <img src="../_static/media/chapter_2/section_1/media/image.png"  class="inline-icon" style="width:400px;" >

2. 将 USB 数据线，一端接入到主控器的充电接口中，另一端接入充电器中。

   <img src="../_static/media/chapter_2/section_1/media/image.png"  class="inline-icon" style="width:400px;" >

3. 充电时主控器上的 LED 灯亮蓝灯，充满后 LED 灯熄灭，充电完成后需及时将电源线拔掉，避免过度充电。

   <img src="../_static/media/chapter_2/section_1/media/image.png"  class="inline-icon" style="width:400px;" >

## 2.5 Program Download Steps

### 2.5.1 Connecting to a Computer

1. 先将主控板的开关拨到“**ON**”档，再将 USB 数据线接入到主控器的程序下载接口中；

   <img src="../_static/media/chapter_2/section_1/media/image.png"  class="inline-icon" style="width:400px;" >

2. 将数据线另一端插入电脑USB卡槽；

   <img src="../_static/media/chapter_2/section_5/media/image2.png"  class="inline-icon" style="width:200px;" >

3. 点击“连接”按钮，并连接对应端口。

   <img src="../_static/media/chapter_2/section_5/media/image3.png"  class="inline-icon" style="width:400px;" >

**注意：连接的端口号不唯一，本小节连接的端口号是“COM3”，但切勿连接“COM1”，它通常为系统通信的接口。如果显示多个COM口无法确定时，可右击电脑的“此电脑”，依次点击“属性->设备管理器”，查看主控器对应的端口号。（含CH340标识即为该端口）**

<img src="../_static/media/chapter_2/section_5/media/image4.png"  class="inline-icon" style="width:400px;" >
