# 4. Standard Kit Creative Builds

## 4.1 Smart Gesture-Controlled Light

### 4.1.1 Project Overview

这是一盏会 “挥手变色” 的智能手控灯，超声波传感器每检测到一次挥手，灯的颜色就切换一次，共有红、绿、蓝和关灯四种状态，就像一位听话的灯光魔术师，用手势就能操控色彩变化。

<img src="../_static/media/chapter_4/section_1/media/image16.png"  class="inline-icon" style="width:400px;" >

### 4.1.2 Learning Objectives

1. 在作品中实践超声波传感器的检测功能，理解手势触发与颜色切换的逻辑。
2. 掌握智能手控灯的机械搭建。
3. 学会编写 “超声波检测 + 颜色切换” 的程序，理解状态切换原理。
4. 提升动手实践与交互设计能力，感受手势控制的科技魅力。

### 4.1.3 Materials Needed

1. **材料准备**：主控器、超声波传感器、模块线、积木零件。

## 4.3 Sound-Activated Ferris Wheel

### 4.3.1 Project Overview

这是一座会 “听声变速” 的声控摩天轮，根据声音传感器检测到的声音次数控制转动速度，转动时主控板 RGB 灯还会持续闪烁，就像游乐园里的欢乐摩天轮，声音越热闹，转得越快，灯光越炫。

<img src="../_static/media/chapter_4/section_3/media/image1.png"  class="inline-icon" style="width:300px;" >

### 4.3.2 Learning Objectives

1. 在作品中实践电机、声音传感器与主控 RGB 灯的基础功能，理解声音检测、速度控制与灯光闪烁的联动逻辑。
2. 掌握声控摩天轮的机械搭建。
3. 学会编写 “声音计数 + 速度切换 + 灯光闪烁” 的程序，理解多设备协同工作原理。
4. 提升动手实践与交互设计能力，感受声控互动的编程乐趣。

## 4.5 Rotary Cutter

### 4.5.1 Project Overview

这是一台会 “触摸启停” 的飞转切割机，触摸传感器检测到触摸时，电机启动带动切割刀片转动；无触摸时，电机立即停止，就像安全的小工匠，只有在操作时才运转，保障使用安全。

<img src="../_static/media/chapter_4/section_5/media/image1.png"  class="inline-icon" style="width:300px;" >
